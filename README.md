> Git 초기설정
>
> ```py
> $ touch README.md #보여주는 파일 만들기

> $ git init #처음에 git장소 부여
> Initialized empty Git repository in C:/Users/multicampus/Desktop/TIL/.git/

>

> user.name=eunbeen0229
> user.email=eunbeen554@gmail.com
>
